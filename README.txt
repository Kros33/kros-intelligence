# Kros Intelligence — Landing Page (Free Hosting)

## Publicar en GitHub Pages (gratis)
1. Crea un repositorio (por ejemplo `kros-intelligence`).
2. Sube los archivos: `index.html`, `styles.css`, carpeta `assets/`.
3. En el repo: Settings → Pages → **Deploy from a branch**, selecciona `main` y carpeta `/ (root)`.
4. Abre la URL `https://<tu-usuario>.github.io/kros-intelligence/`.

## Publicar en Netlify (drag & drop)
1. Entra a https://app.netlify.com/drop
2. Arrastra la carpeta completa `kros_site/`.
3. Netlify asignará un subdominio gratuito (puedes personalizarlo).
4. El formulario de contacto funciona con Netlify Forms gracias a `data-netlify="true"`.
